If extension tooltip doesn't show up:

    check selected type of bubble in Advanced Menu(click on Extension icon in top right corner or use shortcut alt + u),
    there are 4 types (not including "off" state) - google, window, cloud and selection;

    1. if tooltip style set to google, then tooltip should show up,
    otherwise try reloading page(ctrl+r) or re-opening browser.
    else check if Microsoft API key is valid and shown in Main menu.


If extension tooltip doesn't show up (after extension installment):

    1. restart browser - by closing and opening it again.


How to enter API key manually:

    go to rapidapi.com (make sure that you are logged into account)
    on the right side, below "Code Snippets" will be line 'X-RapidAPI-Key': 'YOUR_API_KEY' ,
    copy API key and insert it into Main menu input field.


If button "Get API key (automated)" doesn't work:

    correct behaviour of this button is to make a few website redirects,
    consequently returning to Main menu and filling input field with API key.

    1. If this behaviour doesn't occur, enter API key manually (instructions mentioned above)
